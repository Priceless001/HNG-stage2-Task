* npm
  ```sh
  npm install npm@latest -g
  ```

  * chrome browser

### Installation

1. Get a free API Key at [https://moviedb.org](https://moviedb.org)
2. Clone the repo
   ```sh
   git clone https://github.com/Priceless001/HNG-stage2-Task.git
   ```
3. Install NPM packages
